# Lee Vaakki Dhaba - Product Requirements Document

## Original Problem Statement
Build and improve the Lee Vaakki Dhaba restaurant mobile/web app with Flutter

## Project Overview
A Flutter web/mobile application for an authentic North Indian restaurant (dhaba) that allows customers to browse the menu, add items to cart, and place orders via WhatsApp.

## Tech Stack
- **Frontend**: Flutter (Web & Mobile)
- **Deployment**: Vercel
- **External Integration**: WhatsApp for orders, URL Launcher for maps/phone

## User Personas
1. **Hungry Customer** - Wants to browse menu and order food
2. **Regular Patron** - Knows what they want, quick ordering
3. **First-time Visitor** - Needs info about location, hours, specialties

## Core Requirements (Static)
- [x] Display restaurant menu with categories
- [x] Show item details with prices and descriptions
- [x] Shopping cart functionality
- [x] WhatsApp ordering integration
- [x] Restaurant contact information
- [x] Location and hours display
